1. Cài biến môi trường C:\xampp\php

2. Cài đặt Composer: https://getcomposer.org/download/


3. Thực thi lệnh composer install (bước này hơi lâu nha, phải mạng mạnh chứ không tải lâu,
nên phải làm cái này trước)

    3.1. Mở: C:\xampp\php
    3.2. Kiếm các file có tên: php.ini-development, php.ini-production
    3.3. Mở bằng notepad
    3.4. Tìm và bỏ dấu ; các lệnh sau đây:
        - extension=zip
        - extension=fileinfo
        - extension=mbstring
        - extension=openssl
        - extension=intl       ; (không bắt buộc, nhưng Laravel hay dùng)
        - extension=bcmath     ; (một số package cần)

4. Thêm 1 file .env (vì file này bảo mật nên tui push lên, mọi người pull xuống
cũng không thấy được, nên buộc phải tự tạo)

5. Chạy lệnh  composer require laravel/ui  để cài laravel/ui

6. Chạy lệnh composer require maatwebsite/excel barryvdh/laravel-dompdf spatie/laravel-permission

7. Chạy lệnh composer require league/flysystem-aws-s3-v3:^3

8. Chạy lệnh composer require barryvdh/laravel-dompdf để cài pdf

9. Chạy lệnh composer require maatwebsite/excel:^3.1.55 để cài excel

10. Chạy lệnh composer require laravel/sanctum để cài xác thực (authentication) cho API và SPA

11. Chạy lệnh composer require laravel/socialite : thư viện hỗ trợ OAuth (Google, Facebook, GitHub...)

12. Chạy lệnh  npm install  để cài đặt

13. Khởi động server: php artisan serve

# Clear cache (nếu lỗi view/config)
php artisan cache:clear
php artisan config:clear
php artisan route:clear
php artisan view:clear
